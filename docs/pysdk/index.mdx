The [NeMo Platform Python SDK](https://pypi.org/project/nemo-platform/) is a library for building and deploying AI models, abstracting the underlying infrastructure and providing a high-level interface for the NeMo Platform APIs.

## Installation

Install the PyPI `nemo-platform` wrapper package using `pip`. The wrapper defines the `all` extra and includes the SDK. If you are working from a source checkout, run `make bootstrap` from the repository root instead; the generated SDK package under `sdk/python/nemo-platform/pyproject.toml` does not define the wrapper extras.

```bash
pip install "nemo-platform[all]"
```

<Note>

This project will download and install additional third-party open source software projects. Review the license terms of these open source projects before use.

</Note>
    If you previously installed the `nemo-microservices` package, uninstall it first to avoid conflicts:

    ```bash
    pip uninstall nemo-microservices
    ```
